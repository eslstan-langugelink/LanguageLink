# 2025 Civics Vocabulary Spreadsheet

This folder contains a learner-facing vocabulary spreadsheet built from the official USCIS `2025-Civics-Test-128-Questions-and-Answers.pdf` source.

Files:

- `2025_civics_vocabulary_spreadsheet.csv` — spreadsheet-ready UTF-8 CSV with 250 deduplicated terms and columns for Word, Part of Speech, Simple English Definition, Citizenship Definition, Chinese, Pinyin, Spanish, Example Sentence, Question Number, Audio File, and Approved.
- `2025_civics_vocabulary_spreadsheet.html` — branded HTML spreadsheet view using the Language Link logo, Navy Blue (`#0B1F3A`), and warm Gold (`#D6A64F`). Open this file in a browser or spreadsheet application for a styled review sheet.

Selection criteria:

- Important to understanding the citizenship question or answer.
- Special government, history, geography, or citizenship meaning.
- Likely to be unfamiliar to intermediate English learners.
- Could be misunderstood in civics context.

The `Audio File` column is intentionally blank for later recording or text-to-speech file assignment. The `Approved` column defaults to `No` for instructional review.
